# 🚀 Network Intrusion Detection System (NIDS)

## 📌 Overview
This project is a Machine Learning-based Network Intrusion Detection System that detects malicious network activity.

## 🎯 Objective
To classify network traffic as normal or attack using ML algorithms.

## 📊 Dataset
- Dataset used: [Kaggle]
- Features: duration, protocol_type, service, flag, etc.

## 🧠 Model Used
- AdaBoost / Random Forest / XGBoost  /Bagging Classifier  etc
